# 🏓 Juego de Ping-Pong

¡Bienvenido! Este es mi primer proyecto alojado en GitHub. Es un juego clásico de Ping-Pong desarrollado en Python utilizando la librería gráfica integrada `turtle`.

## 🚀 Características
* Juego clásico para dos jugadores en la misma computadora.
* Contador de puntuación en tiempo real en la pantalla.
* Controles mecánicos fluidos y respuesta rápida.

## 🛠️ Requisitos
Solo necesitas tener instalado **Python 3** en tu sistema. No se requieren librerías externas adicionales.

## 🕹️ Cómo jugar
1. Descarga o clona este repositorio.
2. Abre tu terminal y ejecuta el comando:
   ```bash
   python ping_pong.py
   ```
3. **Controles:**
   * **Jugador 1 (Izquierda):** Teclas `W` (Arriba) y `S` (Abajo).
   * **Jugador 2 (Derecha):** Flecha `Arriba` y Flecha `Abajo`.
